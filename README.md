## Features
- Add, delete, update contacts
- Search contacts by name
- Persistent data storage using file I/O

## Technologies
- C++
- Standard Library (fstream, string, vector)

## How to Run
1. Compile all `.cpp` files together
2. Run the generated executable
3. Data is saved automatically to a text file

## Notes
This project was built to practice modular programming and file operations in C++.
